# video_editer
video mosaic editer

## Pre-requisites

* install libraries
  > pip install requirements.txt
* tensorflow version
  > tensorflow <= 2.4.x

## download weight
[here](https://drive.google.com/drive/folders/1zlSZ9L47BkY0vUKs1n65VRZtAwUvUopG)
